InStorage

Description

InStorage is a web application aimed at solving the storage issues for college students over the summer. We built an online marketplace that allowed students to connect with one another with solutions to their problems. We provide three features: students to store their belongings at more reasonable prices, students to rent out empty spaces in their apartments for extra cash, and students to make money helping others move their items. Because of this, we like to call ourselves Airbnb for storage combined with Uber for moving!

Tools: JavaScript, React, HTML/CSS, Express, Firebase, Jest and Enzyme

Readible

Description

Readible is blog-sharing application that allows users to upload and share their blogs written in Google Docs. The choice of using Google Docs as the editor was to allow for the most accessibility to the users. Anyone can publish their work onto the application by obtaining the embedded link of a Google Doc and submitting it along with a title and author. All of the data is stored in Firebase's Realtime Database. Users can browse through all of the available blogs on the landing page and click to read.

Tools: React, JavaScript, HTML/CSS, and Firebase

Billiards Game

Description

A two-player game rendition of the classic 8-ball pool. The game features two different camera angles: one bird's eye view and another from the perspective of the cue ball. Scratches automatically ends your turn and places the cue ball in the middle of the table. To strike the cue ball, you hit the spacebar and you vary the amount of power used by gauging the distance between the cue stick and ball.

Tools: JavaScript, HTML/CSS, WebGL, and tiny-graphics library

TuneSearch

Description

TuneSearch is a simple search engine for song lyrics. Users can enter a set of words and select either an AND or an OR query. AND means that the results must contain all of the input words, while OR means it may or may not have them. The results are ranked accordingly to its TF-IDF score, which weighs a word's frequency. Each result will contain the title of the song along with its artist.

Tools: HTML/CSS, Python, PostgreSQL, Nginx, Flash, and VirtualBox

Space Blaster

Description

Space Blaster is a spinoff of the classic game Astro Blaster. The mission is to fly your spaceship through outer space, shooting down alien spacecraft and gathering the goodies that they drop. The video game is split into three different phases: initialization, game play, and cleanup. All of the game objects are derived from a single actor base class, with each object implementing its own set of properties such as flying speed, direction pattern, health, amount of damage, etc. A game world class was used to keep track of all objects in the game, including the player, aliens, moving background stars, collectables, game score, and player lives.

Tools: C++

Cracked

Description

Cracked is a decoder for trivially encrypted messages. A tokenizer is used to separate the message. Given a large data set of English words, each encrypted word is cross-referenced against words that follow the same pattern. For matching patterns, the translator class is updated with the pending mapping, which is pushed onto the stack. The decryption is built upon the DFS algorithm, so if there is a conflicting mapping, the program backtracks and pops invalid mappings off the stack until another viable pathway is found. If a dead end is reached, there is no possible translations and nothing is returned.

Tools: C++
